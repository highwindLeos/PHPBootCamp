코드셉트를 이용한 개발 테스트 자동화.

회원가입 절차.
1. 홈페이지 루트로 접근.
2. 이메일 입력 폼에  가입할 이메일을 입력.
3. 성명 입력폼에 이름 입력
4. 사용자 이름 입력폼에 사용자를 입력.
5. 패스워드 입력 폼에 비밀번호를 입력.
6. 가입 버튼을 클릭 해서 입력 완료.
7.메인 페이지가 보이며  데이터베이스 테이블에 입력했던 값들이 행으로 추가되어야 함.

회원 로그인 절차
1. 홈페이지 루트로 접근.
2. 로그인 링크를 클릭
3. 사용자아이디(이메일) 입력 폼에 이메일을 입력함.
4. 패스워드 입력 폼에 비밀번호를 입력.
5. 로그인 버튼을 클릭.
6. 메인페이지가 보이며  사용자 가 쓴 글이 표시됨.

코멘트 달기 절차.
1. 홈페이지 루트로 접근.
2. 로그인 링크를 클릭
3. 사용자아이디(이메일) 입력 폼에 이메일을 입력함.
4. 패스워드 입력 폼에 비밀번호를 입력.
5. 로그인 버튼을 클릭.
6. 메인페이지가 보이며  사용자 가 쓴 글이 표시됨.
7. 코멘트 입력폼에 내용을 입력하고 입력 버튼을 클릭
8. 코멘트 DB 에 값이 있는지를 확인.
